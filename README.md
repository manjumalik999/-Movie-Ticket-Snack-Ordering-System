# -Movie-Ticket-Snack-Ordering-System
🎬 Movie Theater Assistant (C++) 🍿

A beginner-friendly C++ console application that simulates a movie theater experience. The program collects movie details, verifies the ticket number, and allows users to order snacks and drinks before enjoying their movie.

✨ Features
🎥 Enter your movie name
🎟️ Enter ticket number
🍿 Ask whether the user wants food
🥤 Multiple food options:
Popcorn
Drinks
Fast Food
🍕 Different choices for each category
💬 Interactive console-based menu
🧠 Uses conditional statements (if, else if, else)
📥 Takes user input using cin and getline()
🚀 Perfect project for C++ beginners
🛠️ Concepts Used
📌 Variables
📌 Strings
📌 User Input (cin, getline)
📌 Conditional Statements
📌 Nested if-else
📌 Logical Operators (||)
📌 Console Output
🎯 Learning Objective

This project helps beginners understand how to:

Build interactive console applications.
Work with nested decision-making.
Handle user input efficiently.
Create menu-driven programs in C++.
